# FlashMall - 闪购商城

基于 Spring Boot 3 + Vue 3 的分布式闪购商城实战项目。

## 技术栈
- **后端**：Java 17 / Spring Boot 3 / Spring Security / JWT / MyBatis-Plus
- **中间件**：Redis / RabbitMQ / Elasticsearch / Nginx
- **前端**：Vue 3 / Element Plus / Pinia / Axios
- **运维**：Docker / Docker Compose
- **特色**：WebSocket 实时通知 / AI RAG 智能问答
