# FlashMall 技术栈说明


## Java

版本：

Java 17


作用：

后端主要开发语言。



---

## Spring Boot

版本：

Spring Boot 3


作用：

快速构建后端服务。



---

## MyBatis Plus


作用：

数据库ORM框架。

负责：

- 数据访问
- CRUD
- 分页查询



---

## MySQL


作用：

核心业务数据存储。


存储：

- 用户
- 商品
- 订单
- 秒杀记录



---

## Redis


作用：

高性能缓存。


使用场景：

- 商品缓存
- 秒杀库存
- 热点数据



---

## Redisson


作用：

Redis客户端增强。


使用：

- 分布式锁
- Redis工具



---

## RabbitMQ


作用：

消息队列。


使用：

- 秒杀削峰
- 异步订单
- 消息解耦



---

## Spring Security + JWT


作用：

用户认证和权限控制。


---

## Vue3


作用：

管理后台前端。


---

## Docker


作用：

项目容器化部署。



---

## Nginx


作用：

- 反向代理
- 静态资源管理
- 前后端分离部署


---

## Elasticsearch


状态：

计划加入


作用：

商品搜索。



---

## AI RAG


状态：

扩展功能。


作用：

智能商品问答。
