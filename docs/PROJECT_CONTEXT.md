# 开发原则

1. 重复性代码使用AI辅助生成。

例如：

- CRUD
- DTO
- VO
- Mapper
- 基础页面


2. 核心技术自己理解并实现。

包括：

- 系统架构
- 数据库设计
- Redis设计
- 秒杀流程
- MQ异步处理
- 分布式锁
- 性能优化


3. 项目按照真实软件工程流程开发。

包括：

- Git版本管理
- 文档记录
- 代码规范
- 部署上线
