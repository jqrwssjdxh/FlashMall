# 四、项目开发原则


## AI辅助原则


AI负责：

- CRUD代码生成
- 基础页面生成
- 模板代码生成


开发者负责：

- 架构设计
- 技术选型
- 核心业务逻辑
- 性能优化
- 问题分析



## 学习原则


不提前学习全部技术。

按照项目驱动学习。


例如：

开发缓存：

学习Redis。


开发异步：

学习RabbitMQ。


开发部署：

学习Docker。
